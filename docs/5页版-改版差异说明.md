# 5页版改版差异说明

## 1. 改版目标

将原来的 3 页母版演进为 5 页正式版，核心目标是：

- 收敛页面职责
- 降低重复模块
- 保持口径不变
- 提升后续可拆分、可维护能力

## 2. 和 3 页版相比的主要变化

### 2.1 页面结构变化

- 保留：
  - 全国总览
  - 城市经营
- 新增：
  - 时段热力与履约
  - 主体分析
  - 诊断预警
- 兼容：
  - `/direct` 只做跳转，不再作为正式页面

### 2.2 模块归属变化

- 小时运力、热力图、准时率、SLA 迁移到：
  - `/partner/hourly`
- 新骑手、新商家、主体识别、名单明细、骑手提成明细迁移到：
  - `/partner/entities`
- 风险加盟商、关注加盟商、波动预警、健康度详情迁移到：
  - `/alerts`

### 2.3 规则变化

- `merchant_like_threshold` 只在主体分析页生效
- 名单筛选只影响名单模块
- 所有页面单次查询上限统一为 31 天
- `/direct` 不再保留独立业务逻辑

## 3. 不变的内容

- 业务口径不变
- 5 页共享同一套正式主指标
- 直营相关数据接口继续保留
- 旧数据导入与重建方式不变，只增强 `auto / force`

## 4. 验收重点

- 5 个页面都能正常访问
- `/direct` 跳转正确
- 主体识别阈值不再污染名单接口
- alerts 页各模块筛选口径一致
- 页面职责边界清楚，重复模块显著减少
