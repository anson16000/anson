# 5页版页面职责表

| 页面 | 必留模块 | 已移出模块 | 说明 |
|---|---|---|---|
| 全国总览 | 核心摘要、全国趋势、区域排名、加盟商分层、新合伙人表现 | 风险加盟商、关注加盟商、波动预警、健康度详情 | 只做全国总览与排名入口 |
| 城市经营 | 经营摘要、经营收益、直营专项摘要、问题摘要入口 | 热力图详情、名单明细、主体识别、小时运力深度分析 | 成为城市经营首页 |
| 时段热力与履约 | 小时运力、热力图、准时率、SLA 履约率、SLA 超时率 | 主体分析、名单、提成明细 | 专看时段承接与履约 |
| 主体分析 | 新骑手、新商家、主体识别、名单明细、骑手提成明细 | 热力图、全国风险诊断 | 专看骑手、商家、主体结构 |
| 诊断预警 | 风险加盟商、关注加盟商、波动预警、健康度详情 | 全国趋势、区域排名 | 专看风险、预警、整改优先级 |

## 规则补充

- `merchant_like_threshold` 只允许出现在主体分析页
- 名单筛选只影响主体分析页中的名单模块
- 所有页面单次查询最多 31 天
- `/direct` 仅保留兼容跳转，不再作为正式主菜单页面
